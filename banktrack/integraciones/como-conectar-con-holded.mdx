---
title: "Cómo conectar con Holded"
description: "Conecta Holded con Banktrack para sincronizar contactos, facturas y etiquetas."
---

Pulsa en el botón **Añadir canal** para seleccionar la integración de Holded.

![Seleccionar la integración Holded del listado](https://blob-cdn.documentation.ai/org-54c75dd1-b435-4e90-af26-bd7d8846081b/doc-3ced55c8-092e-4386-b871-aab18b35003e/1767001114234-1cibfdq6eiq-CleanShot-2025-03-13-at-13-51-03-2x.png?q=85&fm=auto&auto=compress%2Cformat)

A continuación será necesario generar una API Key en Holded para conectarlo con Banktrack e introducirla en el cuadro de texto siguiente.

![API Key de Holded](https://blob-cdn.documentation.ai/org-54c75dd1-b435-4e90-af26-bd7d8846081b/doc-3ced55c8-092e-4386-b871-aab18b35003e/1767001098351-ljtpil4b42-CleanShot-2025-03-13-at-13-53-39-2x.png?q=85&fm=auto&auto=compress%2Cformat)

Para crear la API Key de Holded puedes seguir los siguientes pasos: Configuración > Más > Desarrolladores > Nueva API Key.

![Generar API Key de Holded](https://blob-cdn.documentation.ai/org-54c75dd1-b435-4e90-af26-bd7d8846081b/doc-3ced55c8-092e-4386-b871-aab18b35003e/1767001186912-nmv1g7q6ypq-Api-key-holded.gif?q=85&fm=auto&auto=compress%2Cformat)

Una vez creada la API Key, la introducimos en Banktrack y le damos a Añadir Holded y Sincronizar. Esta acción sincronizará la información de [Contactos](https://app.banktrack.com/contactos) (Proveedores y Clientes), [Facturas](https://app.banktrack.com/cobros-y-pagos) (Emitidas y Recibidas) y Etiquetas de Holded (aparecerán como una nueva taxonomía en la sección de [Etiquetas](https://app.banktrack.com/etiquetas/) de Banktrack).

Por el momento no está disponible la sincronización automática y será necesario sincronizar manualmente Holded cada vez que queramos actualizar la información.

Si tienes alguna duda con respecto a la integración con Holded puedes contactarnos en el chat que encontrarás en la esquina inferior derecha de la pantalla.
