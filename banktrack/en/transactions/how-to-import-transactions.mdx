## Accepted formats

If you prefer to modify the format of your file without using the template, remember that the formats accepted in Banktrack are the following:

| Field          | Format                        | Required |
| -------------- | ----------------------------- | -------- |
| Concept        | Text                          | ✅ Yes   |
| Amount         | Number without thousands separator | ✅ Yes   |
| Date           | Date in YYYY-MM-DD format     | ✅ Yes   |
| Identifier     | Not necessary                 | ❌ No    |
| Comment        | Text                          | ❌ No    |
| Balance        | Number without thousands separator | ❌ No    |
| Execution date | Date in YYYY-MM-DD format     | ❌ No    |
