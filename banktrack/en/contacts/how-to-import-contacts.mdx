# How to import contacts

To import contacts into Banktrack, you can use the template we provide or adjust your file formatting for proper compatibility.

<Callout kind="info">
  Important: For the import to work correctly, the CSV file must use a semicolon ( ; ) as the separator. Files using a comma ( , ) as the separator are not supported.
</Callout>

## Downloadable template

<Callout kind="tip">
  You can find the downloadable template at this link: [View contact import
  template](https://docs.google.com/spreadsheets/d/126MPJqJbGs8ypYMRs1J_dZIwXEN430lGN57AI8WFWgU/edit?usp=sharing)
</Callout>

## Accepted formats

Take note of the following accepted formats in Banktrack:

| Field        | Format                                                | Required |
| ------------ | ----------------------------------------------------- | -------- |
| Name         | text                                                  | ✅ Yes   |
| Contact type | `freelancer` (self-employed) or `company`             | ❌ No    |
| Fiscal ID    | text and/or numbers                                   | ❌ No    |
| Email        | text and/or numbers                                   | ❌ No    |
| Phone        | text and/or numbers                                   | ❌ No    |
| Address      | text and/or numbers                                   | ❌ No    |
| Postal code  | text and/or numbers                                   | ❌ No    |
| City         | text and/or numbers                                   | ❌ No    |
| Province     | text and/or numbers                                   | ❌ No    |
| Country      | ISO code (e.g., Spain = `ESP`, Mexico = `MEX`)        | ❌ No    |

If you have any questions regarding importing contacts, you can contact us through the chat found in the bottom right corner of the screen.
